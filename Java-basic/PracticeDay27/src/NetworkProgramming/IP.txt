IP(Internet Protocol):互联网协议地址,也称IP地址,是分配给上网设备的数字标签,也就是上网设备在网络中的地址,是唯一的
常见的IP分类为:IPv4和IPv6


IPv4(Internet Protocol version 4):互联网通信协议第四版,采用32位地址长度,分为4组.于2019年11月26日分配完毕
由于互联网的蓬勃发展,IP地址的需求量愈来愈大,IPv4模式下IP的总数是有限的,故推出了IPv6协议
IPv6(Internet Protocol version 6):互联网通信协议第六版,采用128位地址长度,分为8组


IPv4的地址分类形式:
1.公网地址(万维网使用)
公网地址是指在因特网上直接可达的地址,如果你有一个公网地址,那就意味着你不但能访问别人,还能被人访问.公网地址是架设各种服务器必不可少的条件
公网地址被分为了五类:
  A类:10.0.0.0 -- 127.255.255.255 主要分配给大量主机而局域网网络数量较少的大型网络
  B类:128.0.0.0 -- 191.255.255.255 一般用于国际性大公司和政府机构
  C类:192.0.0.0 -- 223.255.255.255 用于一般小公司校园网研究机构等
  D类:224.0.0.0 -- 239.255.255.255 用于特殊用途，又称作广播地址
  E类:240.0.0.0 -- 255.255.255.255 暂时保留

2.私有地址(局域网使用)
IPv4地址中预留了3个IP地址段来作为私有地址,供家庭、企业、学校等内部组网使用.分为三个ip地址段,除了这三个ip地址段为私有ip地址外,其它的都为公网ip:
  第一段:10.0.0.0 -- 10.255.255.255
  第二段:172.16.0.0 -- 172.31.255.255
  第三段:192.168.0.0 -- 192.168.255.255
任何在此范围内的IP地址都可以被任何组织或个人在其内部网络中自由使用而无需向全球互联网注册机构申请
每个私有IP地址可以在任何数量的网络中重复使用,因为这些网络从不直接相互通信(指的是网络与网络之间不直接相互通信,但通过网络地址转换（NAT）后可以用于互联网通信)
其中192.168.开头的是专门为组织机构内部使用,以此节省IP

私有IP地址主要用于本地网络,包括家庭,学校和办公室的局域网.大多数Wi-Fi路由器都被配置为从一个私有IP地址范围(通常是192.168.0.0/16)分配地址。

当从私有IP地址进行的设备需要访问互联网时会使用一种名为网络地址转换(NAT)的技术
NAT允许路由器将许多私有IP地址映射到一个或多个公共IP地址.这是实现单个互联网连接服务于整个家庭或办公室网络的方式


特殊IP地址:
localhost(127.0.0.1):回送地址也称本地回环地址,也称本机IP,永远只会寻找当前所在本机.所有计算机都使用此地址作为自己的地址,但它不会像真实IP地址那样让计算机与其他设备通信
实际上网段127.0.0.0/8中的任何IPv4地址都被指定为环回地址

假设192.168.1.100是我电脑的IP,那么这个IP跟127.0.0.1是一样的吗?
  1.192.168.1.100属于私有IP地址,通常用于家庭或企业内部局域网,需要通过路由器的NAT功能转换为公网IP才能访问互联网
    如果计算机设备的真实方位发生了变化,该IP地址可能也会随之发生变化
  2.127.0.0.1是本地回环地址,属于保留地址,代表计算机本身,用于本地网络服务的测试(如访问本地运行的Web服务器,故以后练习时自己为自己发送数据可以使用这个IP)
    发送到这个地址的数据不会经过物理网卡,直接在本地协议栈处理
  因此192.168.1.100是个人计算机设备在局域网中的真实地址,用于与其他设备通信
     而127.0.0.1是本地测试用的虚拟地址,仅仅代表自己的电脑
     两者并不是相同的概念


常用的CMD命令:
1.ipconfig:查看本机IP地址
2.ping:检查网络是否连通
